# Snow Preferences Template

Copy to `memory/snow-preferences.md` and customize.

```markdown
# Snow Preferences

## Default Mountain
jacksonhole

## Favorites
- jacksonhole
- grandtarghee
- alta

## Report Style
compact

## Skip
- parking
```

## Configuration Options

### Default Mountain
Single OpenSnow slug. Used when user says "snow report" without specifying.

### Favorites
List of slugs for comparison reports and powder alerts. Keep to 3-5 for readable tables.

### Report Style
- `compact` — Single line snow + conditions + brief expert take
- `detailed` — Full table with all forecast windows + complete narratives

### Skip
Data to omit from reports:
- `parking` — Skip parking availability
- `events` — Skip resort events
- `ai` — Skip AI overview (just show Daily Snow expert)
